# data-analytics-portfolio7
## Project 07 - World Data SQL Analysis

### Tools Used
- DB Browser for SQLite
- SQL (CREATE, INSERT, SELECT, WHERE,
  GROUP BY, ORDER BY, SUM, AVG, HAVING,
  CASE WHEN)

### Database Structure
- Table : countries (15 records)
  id, country, continent, population,
  gdp_per_capita, life_expectancy, literacy_rate

### Key Findings

1. Life Expectancy by Continent :
   Europe has the highest average life expectancy
   (81.6 years) due to advanced healthcare systems
   and high development levels.
   Africa has the lowest (64.8 years) due to
   poverty and lack of medical infrastructure.

2. Population Analysis :
   Asia concentrates 3.2 billion people —
   nearly 3x more than all other continents
   combined in our dataset.

3. Economic Classification :
   - Rich countries (GDP > 20,000) : USA, Germany,
     France, Japan, UK, Canada
   - Middle countries (GDP > 5,000) : China, Brazil,
     South Africa, Mexico
   - Poor countries : Nigeria, India, Egypt,
     Indonesia, Ethiopia

4. Continents with population over 500 million :
   Asia, America and Africa.

### Recommendation
African governments should prioritize healthcare
infrastructure investment — building equipped
hospitals and medical centers would directly
improve life expectancy and quality of life
across the continent.
